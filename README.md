## 1. 웹 서비스 소개

셜록 옷즈
 무신사를 활용한 패션 정보 제공 및 정보 교류 서비스

선정배경: 패션능력치 향상을 위해 들어야하는 노력이 많으므로, 이에 대해 도움을 주고자함

문제정의: 패션 조합의 톤앤매너를 맞추기 어려우므로, 이를 도와주기 위해 유사한 아이템 소개, 색조합 평가, 피어 평가등이 필요함.
